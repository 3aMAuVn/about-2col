Booking / enquiry system notes
=============================

What is included
- Shared premium booking hub at /booking.html
- Existing registration and enquiry pages preserved and merged into Jekyll layout
- Data-driven navigation and footer CTAs

How to make it fully live
1. Keep the current forms if you already have a working backend.
2. Or connect the forms to Formspree / Netlify Forms / Basin.
3. Or replace /booking.html CTA links with a scheduler such as Calendly or TidyCal.

Recommended next config point
- Edit _data/site.yml to update email, booking_url, phone, and any external scheduler link.
